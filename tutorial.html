PTT Hate Speech Tagger

使用教學

快速開始

下載 Chrome 插件

  1. 打開 https://github.com/celestine1305/PTTHateSpeechTagger/tree/cele
  2. 點選右上角綠色 Code -> Download ZIP
  3. 解壓縮 PTTHateSpeechTagger
  4. 在 Chrome 瀏覽器中打開 chrome://extensions/
  5. 打開右上角 開發人員模式，選擇 載入未封裝項目
  6. 選擇剛剛下載的檔案中的 chrome-extension 資料夾，按 選取

開始標註

  1. 必須在 ptt.cc 這個 domain 上才能使用
  2. 使用 Chrome 瀏覽器，點選右上角的拼圖，打開 PTT Hate Speech Tagger，並把 Enable Toolkit 的滑桿打開
  3. 會跳出一個視窗詢問你的 Token，可以先按取消來關閉
  4. 點選 標註這篇文章，按下 確定 即可標註內文
  5. 點選留言，按下 確定 即可標註該留言

查看排行榜

  1. 點選 排行榜 http://localhost:8080/rank.html
  2. 搜尋欄留空，即為目前所有被標記 ID 的前十名
  3. 在想要查看的 ID 上點一下
  4. 點選 查詢此帳號即可查看該帳號被標注的言論

黑名單查詢

  1. 點選 黑名單查詢 http://localhost:8080/search.html
  2. 輸入想要查詢的 PTT ID
  3. 點選推文紀錄即可查看該篇文章

申請個人帳號

若沒有申請個人帳號，則會使用公用的 Hypothesis 帳號來標註

申請 Hypothesis Token

  1. 打開 https://web.hypothes.is/start/
  2. 點擊 Create A Free Account
  3. 申請帳號，並到 email 點擊認證連結
  4. 認證過後，點擊 Setting > Develper

  5. 點擊 Generate your API token，並將 token 複製起來

  6. 點擊 Extension 的 Icon，並點選 Change Token

  7. 輸入複製的 token 即完成

查看我的排行榜

  1. 點選 排行榜 http://localhost:8080/rank.html
  2. 搜尋欄輸入自己的 Hypothesis ID，即為目前所有被自己標記 ID 的前十名
